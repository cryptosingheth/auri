# Auri — Real gold, made usable.

Marketing website for Auri: buy, borrow against, spend, gift and send real audited gold — in your own custody. Live in Canada.

- `index.html` — main site (self-contained: fonts embedded, assets relative)
- `docs.html` — documentation (getting started, borrowing, card, MCP/CLI, API, custody)
- `clips/` — lifestyle montage + section clips (see PROMPT-PACK.md for regenerating better ones)
- `assets/shots/` — real product screens
- `brand/` — Auri logo system ("Dawn tittle" wordmark + alternates)

Deployed with GitHub Pages.
